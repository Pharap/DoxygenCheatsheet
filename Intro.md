# Intro

## Comment Syntax

Style 1:
```cpp
///
/// Comment
///
```

Style 2:
```cpp
//!
//! Comment
//!
```

Style 3A:
```cpp
/**
 * Comment
 */
```

Style 3B:
```cpp
/**
Comment
*/
```

Style 4A:
```cpp
/*!
 * Comment
 */
```

Style 4B:
```cpp
/*!
Comment
*/
```

## Command Styles

Style 1:
```cpp
/// @brief Brief description
```

Style 2:
```cpp
/// \brief Brief description
```
